# DoNext_web
A dark minimalist task management web app focused on clarity, speed, and intentional design.
DoNext is a dark minimalist task management web application designed around clarity, speed, and intention.  
It removes unnecessary complexity and focuses on what matters most — helping you get things done without distraction.

---

##  Philosophy

DoNext follows a simple principle: **less, but better**.

- Clean, distraction-free interface  
- Thoughtful visual hierarchy  
- Fast, predictable interactions  
- Zero learning curve  

The design stays out of your way so your focus stays on the work.

---

##  Design System

- **Theme:** Dark Minimalist / Premium SaaS
- **Background:** Pure black (`#000000`)
- **Surfaces:** Deep gray cards (`#121212`, `#1C1C1E`)
- **Layout:** Bento grid–inspired modular blocks
- **Components:** Heavy rounded corners (squircles)
- **Navigation:** Floating bottom navigation with glassmorphism
- **Typography:** Modern sans-serif with strong contrast
- **Shadows:** None (flat design)

---

##  Features

- Add, edit, and delete tasks
- Mark tasks as completed
- Task categorization and visual tags
- Due dates for better planning
- Basic task filtering (All / Active / Completed)
- Calendar-style activity visualization
- Persistent data using `localStorage`
- Fully responsive layout

---

##  Privacy by Design

All tasks are stored **locally in your browser**.  
No accounts. No tracking. No servers.

Your data never leaves your device.

---

##  Tech Stack

- **HTML5** — semantic structure
- **CSS3** — external stylesheet with CSS variables for theming
- **Vanilla JavaScript** — clean, modular logic
- **localStorage** — client-side persistence

No frameworks. No dependencies.
